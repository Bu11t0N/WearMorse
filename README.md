# WearMorse Messenger 📡⌚

A fun, lightweight open-source messenger system for Android and Wear OS that allows communication via Morse code and real-time audio monitoring.

## 🚀 Features
* **Phone to Watch:** Send text messages from your smartphone directly to your Wear OS device, translated into Morse code vibrations and on-screen text.
* **Covert Audio Streaming:** Stream audio from the watch's microphone straight to the phone's speaker in real-time.
* **100% Standalone & Local:** Works directly via Bluetooth/Wearable API without external servers.

## 🛠 Project Structure
* `:app` — Android smartphone application (manages UI input and audio playback).
* `:wear` — Wear OS smartwatch application (handles Morse vibrations and audio recording).
